# 단어 학습 플래시 카드 앱

## 프로젝트 개요
단어 학습을 위한 플래시 카드 웹 애플리케이션으로, 4지선다형 퀴즈 형태로 단어와 의미를 학습할 수 있습니다.

## 주요 기능
- 📚 단어 파일에서 데이터 로드 (js/words.js)
- 🎯 4지선다형 퀴즈 형식
- 📱 PC와 모바일 반응형 디자인
- 📝 한 세션당 10개 단어 학습
- 📊 학습 점수 기록 및 조회
- ✨ 모던한 UI/UX 디자인

## 파일 구조
```
index.html          # 메인 HTML 파일
css/style.css       # 스타일시트
js/words.js         # 단어 데이터 및 점수 관리
js/quiz.js          # 퀴즈 로직
```

## 사용 방법
1. `index.html` 파일을 웹 브라우저에서 열기
2. "학습 시작" 버튼 클릭
3. 10개의 단어 퀴즈 풀기
4. 점수 확인 및 기록 저장
5. "점수 보기" 버튼으로 이전 점수 조회

## 디자인 특징
- 그라데이션 배경
- 흰색 카드 UI (welcome-card, flash-card 등)
- 부드러운 애니메이션 효과
- 모바일 우선 반응형 디자인

## 최근 업데이트
- ✅ 점수 기록 및 조회 기능 완전 구현
- ✅ localStorage를 이용한 점수 저장
- ✅ 테스트 점수 자동 추가 기능
- ✅ welcome-card에 흰 박스 배경 추가
- ✅ 개선된 시각적 효과 및 그림자 효과
- ✅ 향상된 모바일 반응성

## 알려진 이슈 및 해결사항
- **점수 보기 버튼 클릭시 점수가 보이지 않는 문제** → 해결됨
  - localStorage 점수 저장/불러오기 기능 구현
  - 테스트 점수 자동 추가로 초기 사용자 경험 개선
  - 점수 데이터 유효성 검사 추가
